# Archive Index

`docs/archive/` 保存历史版本文档、阶段计划、过程记录、旧 runbook 和阶段性交付记录。这里的内容默认不再作为当前实施依据。

## 目录说明

- `history-v1/`
  v1 维护期文档与当时的事实源。

- `history-v2/`
  v2 阶段设计、协议、阶段验收和迁移记录。

- `plans/`
  路线图、开发日志、阶段计划、实现计划。

- `process/`
  工程流程、Git 工作流、发布流程、第三方治理、旧 ADR。

- `releases/`
  历史发布说明与阶段性交付记录。

- `runbooks/`
  已退出主维护面的旧运行/部署/观测手册。

## 使用规则

- 归档文档仅用于追溯背景，不作为当前事实源。
- 当前项目状态、可执行入口、验证口径，以 `../current-state.md` 为准。
- 当前主线文档索引，以 `../README.md` 为准。
