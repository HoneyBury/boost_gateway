# v1.2.5 发布说明

发布日期：2026-05-08

## 发布定位

`v1.2.5` 是 `v1.x` 维护期收束后的稳定性补丁版，不引入 `v2.0.0` 范畴的新结构开发。

## 本版完成内容

- CI：稳定 Windows runner 行为，避免依赖固定环境状态
- Docker：修正依赖获取与 Boost 压缩包 locale 处理
- 测试：提高 GoogleTest discovery timeout，降低 CI 偶发超时
- 分支流转：将上述稳定性修复从 `main` 回灌到 `develop`

## 工程影响

- 不改变 `v1.x` 能力成熟度口径
- 不改变 `v1.2.0 / T21` 的结构升级决策
- `develop` 仍是 `v1.x` 日常集成分支，不视为已进入 `v2.0.0`

## 验证口径

- 本版主要面向发布链路、CI 环境和安装打包稳定性
- 业务主链与 `v1.2.4` 一致，仍以既有回归测试基线为准
