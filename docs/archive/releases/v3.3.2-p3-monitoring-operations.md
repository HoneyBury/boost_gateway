## 当前边界

- 当前 `/metrics` 没有 route latency histogram/summary，因此默认 Prometheus rules 不伪造 P99 告警；P99 仍由性能基线和后续 metrics 扩展沉淀。
- RSS/fd 告警依赖 process exporter 或等价 agent；默认 Compose 只 scrape gateway `/metrics`。
- Redis down 通过 leaderboard backend 的 gateway RED counters 和日志进行生产判断；Redis exporter 可作为后续增强。
