## 波动收束点

- `examples/v2_gateway_pressure/main.cpp`：压力工具 stop 流程改为在 `io_context` 线程内统一收口，避免 duration 线程直接操作 clients/timers。
- `examples/v2_gateway_pressure/main.cpp`：增加 stop grace timer，异常情况下有界退出，正常路径保持 `forced_timeout=false`。
- `include/v2/gateway/gateway_service_bridge.h`、`src/v2/gateway/gateway_service_bridge.cpp`：gateway bridge 增加 backend connection pool、round-robin 和失败重试，降低高并发单连接瓶颈和瞬时读写失败波动。
- `include/v2/service/backend_connection.h`、`src/v2/service/backend_connection.cpp`：backend connection 访问加锁，避免并发请求共享连接时的读写状态竞争。
- `scripts/collect_v2_perf_baseline.py`：Release baseline gate 记录 `forced_timeout`、rejected/failed、battle 最小消息数和 p99 阈值，失败时显式阻断。
