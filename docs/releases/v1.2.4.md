# v1.2.4 发布说明

发布日期：2026-05-07

## 发布定位

`v1.2.4` 是 `v1.x` 维护期的阶段收束版，不引入 `v2.0.0` 范畴的新结构开发。

## 本版完成内容

- `T18`：治理边界测试加固
- `T19`：生命周期与装配测试加固
- `T20`：持久化 / 审计 / 回放测试加固
- `T21`：结构升级决策记录落档，结论为当前维护分支不进入 `v2.0.0`

## 工程化发布补充

- GitHub Actions `CI` 覆盖 Linux / macOS / Windows
- GitHub Actions `Release` 按 tag 构建、测试、安装、打包并上传 release asset
- 安装包统一携带运行配置和维护文档，便于归档与交付
